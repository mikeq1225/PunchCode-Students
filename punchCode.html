    10 things you should study before starting class
  1. Get very comfortable in Terminal

    You spend a lot of time in terminal. Getting comfortable adding or removing files and directories, being able to traverse the file system,and knowing hot to use git will be invaluable to you.

  2. Take a tutorial on VS Code.

    There are a ton of shortcuts in VS Code that will make your life easier if you know how to use them.
    You don't have to be an expert, but if you can master some of the keyboard shortcuts, you will be more efficient in writing and editing your code.
